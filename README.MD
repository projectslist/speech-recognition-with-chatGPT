# ChatGPT - AI Chatbot
### Overview
This project uses GPT-3 for natural language processing and OpenAI's API for creating an AI chatbot. The Chatbot will respond to user input and attempt to hold a conversation. The chatbot can be used to answer general questions or for having casual conversations.

## Dependencies
This project depends on the following Python packages:

openai - for using OpenAI's API for creating an AI chatbot
gradio - for creating the user interface
whisper - for audio processing
pyttsx3 - for text-to-speech conversion

# Usage
- Add your openai API key into config.ini
- Run the following command to start the chatbot:
- ``python speech-recognition-with-chatGPT.py``
